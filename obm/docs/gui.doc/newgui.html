IRAF Graphical User Interfaces

This hypertext work is introductory documentation for the IRAF graphical user interface facility introduced with IRAF version 2.10.3. Questions can be directed, via electronic mail, to iraf.noao.edu or posted to the newsgroup adass.iraf.programmer.
